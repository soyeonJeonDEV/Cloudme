# 구르美(Cloudme)

### 멀티캠퍼스 '프로젝트형 클라우드 서비스 개발' 전공 프로젝트

- 진행기간 : 2022.04.11 ~ 2022.05.02

### 주제
- 여행객을 위한 코로나 정보 웹 사이트

### 팀명
- 구르美

### 팀원
조민기(팀장), 김동현, 유상수, 전소연
